# Bash Scripts

Handy bash scripts for developer productivity.

## Scripts
- `backup.sh` - Automated backup utility
- `deploy.sh` - Simple deployment script
- `cleanup.sh` - System cleanup tool
- `monitor.sh` - System resource monitor
- `setup.sh` - Dev environment setup

## Usage
```bash
chmod +x scripts/*.sh
./scripts/backup.sh /path/to/dir
```
